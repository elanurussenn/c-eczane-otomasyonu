# C Eczane Otomasyonu

Bu proje, C programlama dili kullanılarak geliştirilmiş bir konsol tabanlı eczane otomasyonu uygulamasıdır.

Uygulama; hasta kayıt, hasta arama, ilaç kayıt, ilaç arama, hasta/ilaç silme, güncelleme, reçete bilgisi ve satış işlemleri gibi temel eczane yönetim işlemlerini içermektedir.

##  Projenin Amacı

Bu proje; C programlama dilinde struct yapıları, pointer kullanımı, dinamik bellek yönetimi, fonksiyonlar, koşul yapıları ve döngüler gibi temel konuları pekiştirmek amacıyla geliştirilmiştir.

## Kullanılan Teknolojiler

* C Programlama Dili
* GCC Compiler
* Code::Blocks IDE

##  Özellikler

* Hasta kayıt işlemi
* Hasta arama işlemi
* Hasta silme ve ekleme işlemleri
* Hasta listeleme
* İlaç kayıt işlemi
* İlaç arama işlemi
* İlaç silme ve ekleme işlemleri
* İlaç listeleme
* Satış yapma işlemi
* Satış bilgisi görüntüleme
* Reçete bilgisi işlemleri
* Hasta sağlık uygunluk testi
* Hasta ve ilaç bilgisi güncelleme

##  Dosya Yapısı

```txt
c-eczane-otomasyonu/
├── main.c
└── README.md
```

##  Çalıştırma

Terminal veya komut satırında:

```bash
gcc main.c -o eczane
./eczane
```

Windows için:

```bash
gcc main.c -o eczane.exe
eczane.exe
```
